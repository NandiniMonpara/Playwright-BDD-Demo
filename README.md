# playwright-bdd-demo

Example project that uses [playwright-bdd](https://github.com/vitalets/playwright-bdd) to run BDD tests with [TestDino](https://testdino.com/) reporting.

## Get Started

1. Clone the repo

   ```
   git clone https://github.com/YOUR_GITHUB_USERNAME/playwright-bdd-demo.git
   ```

2. Change directory

   ```
   cd playwright-bdd-demo
   ```

3. Install dependencies

   ```
   npm install
   ```

4. Install browsers

   ```
   npx playwright install
   ```

5. Run tests

   ```
   npm test
   ```

   Output:

   ```
   Running 3 tests using 1 worker
   3 passed (8.5s)
   ```

Results stream live to your [TestDino dashboard](https://app.testdino.com).
